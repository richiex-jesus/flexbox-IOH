Images

This page covers common image formats, but you may also be looking for links and useful extras

JPGs

JPG images are good for photos. Thye are designed for handling large color palettes without substiantially increasing the size of the file, making them great for photos with a lot of colour gradients.

GIFs

GIFs are good for animations. But, their abilities for color paletting is somewhat limited, so never use them for photos.

PNGs

PNGs are good for diagrams or icons. A PNG file of the same quality would generally be bigger than that of an equivalent JPG file. They deal with opacity just fine and also lack color palette limitations, making them excellent fit for icons, technical diagrams, logos etc.

SVGs

SVG is a vector-based graphics format, meaning that it can scale up or down to any dimension without loss of quality, making SVG a great tool for responsive design.

For them to display consistently, you need to convert any text fields to outlines using image editor. If images contain a lot of texts, this can have a big impact
